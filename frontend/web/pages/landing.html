— Возможности

Всё, что нужно
для общения

01

Мгновенные сообщения

WebSocket-соединение — сообщения появляются у всех участников в реальном времени, без перезагрузки.

02

Безопасная авторизация

Сессионная аутентификация, bcrypt-хеши паролей. Никто посторонний не войдёт.

03

Групповые чаты

Создавай комнаты, приглашай людей по публичному ID, управляй участниками.

04

Поиск чатов

Найди нужную комнату за секунду — просто введи публичный идентификатор.

05

REST API

Полноценный FastAPI-бэкенд с авто-документацией. Подключай ботов, свои клиенты, интеграции.

06

Открытый код

MIT-лицензия. Разворачивай на своём сервере, форкай, вноси свой вклад.

— Как это работает

Четыре
простых шага

I

Регистрация

Создай аккаунт за несколько секунд — только логин и пароль, ничего лишнего.

II

Создай чат

Открой новую комнату или найди существующую по публичному ID.

III

Собери людей

Поделись ID или ссылкой. Твоё сообщество присоединится в один клик.

IV

Общайся

Сообщения мгновенно. Никакой слежки, никакой рекламы. Только люди.
